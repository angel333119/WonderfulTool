--- PORTUGUÊS ---

COLOQUE A FONTE AQUI

Para que o programa utilize uma fonte personalizada no previsualizador, por favor, coloque o arquivo da fonte (com a extensão .ttf, .otf, ou .ttc) DENTRO DESTA PASTA.

- É recomendado usar apenas UM arquivo de fonte.
- Se mais de um arquivo for colocado aqui, o programa carregará o primeiro que encontrar (em ordem alfabética).
- Se nenhum arquivo de fonte for encontrado, o programa usará uma fonte padrão do sistema.

--- ENGLISH ---

PLACE THE FONT FILE HERE

To have the program use a custom font in the previewer, please place the font file (with a .ttf, .otf, or .ttc extension) INSIDE THIS FOLDER.

- It is recommended to use only ONE font file.
- If more than one file is placed here, the program will load the first one it finds (in alphabetical order).
- If no font file is found, the program will use a default system font.
